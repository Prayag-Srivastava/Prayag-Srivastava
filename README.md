## 🔥 What I'm Currently Building

### [PitchProbe](https://github.com/Prayag-Srivastava/pitchprobe) - Multi-Agent AI Research Platform *(In Progress)*

> An AI-powered platform that automates startup, market, and competitor research using autonomous agents.

**What makes it interesting:**
- 🧠 **Multi-agent orchestration** using LangGraph with supervisor-worker architecture
- 🔍 **RAG pipelines** over uploaded pitch decks using ChromaDB + vector embeddings
- 🛠️ **Tool calling** — agents autonomously decide when to search the web, retrieve documents, or reason
- ⚡ **Real-time streaming** via FastAPI + WebSockets so you see agent reasoning live
- 🧱 **Production features** — persistent memory, human-in-the-loop approvals, guardrails, LangSmith observability

**Tech Stack:**
`Python` `LangChain` `LangGraph` `FastAPI` `PostgreSQL` `ChromaDB` `Redis` `WebSockets`
